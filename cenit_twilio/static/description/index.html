Setting up...

First things first: SignUp with Twilio

Go to your Account Settings

Take note of your Account SID and Authorization Token

That's pretty much it

After installing the addon a form will let you set the data

Featuring...

Data definitions

Various Schemas that will allow you to map your Odoo models as Twilio's.

  • Message

Set ups

Several Webhooks where you can send or get data.

  • Get List SMS
  • Get SMS by SID
  • Send SMS

A handful of Events and Translators to decide when and how perform transformations.

Events:

  • Message on created_at
  • Get Message

Translators:

  • Export Message
  • Import Message
  • Get Message by ID
  • Import Response

Out of the Box Flows so you can focus on what's really important.

  • Send Message
  • Get Message by ID
  • Gets Message List

First steps

There's really only one thing you need to do: Map your models

  • Go to the Cenit Integrations menu.
  • Create a new Data type
    • Name it as you will.
    • Select the model you wish to send over to Twilio (say 'Message').
    • Select the 'Twilio ' library
    • and the specific schema you want to use (say 'message.json ').
    • On the Mappings tab set how you want your messages to translate to Twilio
    • Save your work.

That's it! Everytime you create a message in Odoo, a message will be create on your Twilio account.

For more on mapping check the Documentation tab of the Cenit Integrations Client.
